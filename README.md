# CRUD - Assignment Q4 (__q4)

A CRUD using quasar framework

## Install the dependencies
```bash
$ yarn install
```

### Start the app in development mode (hot-code reloading, error reporting, etc.)
```bash
$ quasar dev

# Progressive Web App
$ quasar dev -m pwa
```


### Build the app for production
```bash
$ quasar build

# Progressive Web App
$ quasar build -m pwa
```
